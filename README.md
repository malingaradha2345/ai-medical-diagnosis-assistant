# AI Medical Diagnosis Assistant

## Overview

AI Medical Diagnosis Assistant is a machine learning web application developed using Streamlit. It predicts the likelihood of multiple diseases based on user-provided medical parameters.

## Features

- Heart Disease Prediction
- Diabetes Prediction
- Liver Disease Prediction
- Parkinson's Disease Prediction
- Kidney Disease Prediction
- Confidence Score for Predictions
- User-friendly Streamlit Interface

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- XGBoost
- Joblib

## Installation

1. Clone the repository

```bash
git clone https://github.com/malingaradha2345/ai-medical-diagnosis-assistant.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
